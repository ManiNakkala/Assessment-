### Setup Instructions

1. Clone the repo and navigate into backend:
```bash
cd backend
```

2. Install dependencies:
```bash
npm install
```

3. Setup your `.env`:
```bash
cp .env.example .env
```

4. Make sure MongoDB is running locally (or provide a MongoDB Atlas URI in `.env`)

5. Run the backend:
```bash
npm run dev
```

Test the APIs using Postman or any client of your choice.
